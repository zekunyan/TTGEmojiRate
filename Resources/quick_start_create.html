01 / Vertical feeling input

Let users rate by moving emotion

The core interaction is direct: drag the emoji up or down to turn a feeling into a score. Free-step values keep the motion smooth.

let rateView = EmojiRateView(frame: frame)
rateView.maximumRateValue = 10
rateView.rateStep = 0
rateView.rateGestureDirection = .vertical

rateView.rateValueChangeCallback = { score in
    updateFeedback(score)
}
