02 / Shape, fill, and gradient

Make each rating surface distinct

Choose a face shape, turn on fill, and combine dynamic colors with gradient strokes so the control can match different product contexts.

rateView.faceShape = .blob
rateView.faceShapeMorphEnabled = true
rateView.faceFillEnabled = true

rateView.rateGradientColorEnabled = true
rateView.rateGradientColors = [
    .systemCyan, .systemBlue
]
