03 / Replace or overlay the drawing

Draw product-specific emoji paths

Use a replacement path for branded symbols, waveform reactions, or delegate-driven shapes. Replacement paths can include the face outline and inner expression together.

rateView.customEmojiPathMode = .replace
rateView.customEmojiPathProvider = { rect, progress, view in
    let path = UIBezierPath()
    path.append(faceOutline(in: rect))
    path.append(innerSymbol(in: rect, progress: progress))
    return path
}
